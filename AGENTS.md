# AGENTS.md — chatting_app

## Commands

- `flutter run` — launch on connected device/emulator
- `flutter analyze` — static analysis / lint (uses `flutter_lints` ruleset)
- `dart format .` — format all Dart files
- `flutter test` — run tests
- `flutter build apk/ios/web` — production builds per platform

## Architecture

| Layer | Location | Notes |
|---|---|---|
| Entrypoint | `lib/main.dart` | `Firebase.initializeApp` → `MultiBlocProvider` → `MaterialApp` with named routes |
| State | `lib/cubits/` | flutter_bloc Cubit pattern; 3 cubits: `LoginCubit`, `RegesterCubit` (stub), `ChatCubit` |
| Screens | `lib/screens/` | `log_in.dart`, `resgister_screen.dart`, `chat_screen.dart` |
| Widgets | `lib/widgets/` | Reusable: `ChatBuble`/`ChatBubleForFriend`, `CustomButton`, `CoustomTextField` |
| Model | `lib/models/message.dart` | `Messages` class with `fromJson` — Firestore collection `Messages` |
| Constants | `lib/consntant.dart` | Colors, asset paths, Firestore field keys |

## Routes (defined in `main.dart`)

| Key | Route | Screen |
|---|---|---|
| `'login Screen'` | initial | `LogInScreen` |
| `'register'` | named | `RegisterScreen` |
| `'chatScreen'` | named | `ChatScreen` (receives email via route arguments) |

## Firebase

- Configured via `lib/firebase_options.dart` (generated by FlutterFire CLI)
- Supports: Android, iOS, macOS, Web
- **Not configured**: Windows, Linux (throw `UnsupportedError`)
- Firestore collection `Messages` with fields: `message`, `createAt`, `id`
- Auth: email/password via `firebase_auth`

## Repo quirks

- `lib/consntant.dart` misspelled (not `constant.dart`) — import path is `package:chating/consntant.dart`
- `lib/screens/resgister_screen.dart` misspelled (not `register_screen.dart`) — import path `package:chating/screens/resgister_screen.dart`
- `LoginCubit` state uses typo `LoginLoding` (missing `a`)
- `RegesterCubit` is empty (no registration logic — registration is handled inline in `RegisterScreen`)
- `test/widget_test.dart` is a stale default smoke test that references a non-existent counter — do not rely on it
